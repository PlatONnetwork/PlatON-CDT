# PlatON-Test


## Introduction

PlatON-Test is C++ Test tools

### compile

```
platon-test compile --dir {test file dir} | --file {test file}  --output {output dir} --bin {platon-cpp path}
```

### exec

```
platon-test exec --dir {wasm test file dir} | --file {wasm test file} 
```
